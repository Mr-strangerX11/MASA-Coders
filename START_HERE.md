## 🚀 3-Step Quick Start

### Step 1: Install & Setup (5 minutes)
```bash
cd /Users/macbook/Desktop/web
npm install
```

Create `.env.local`:
```
MONGODB_URI=mongodb://localhost:27017/nexuspro
JWT_SECRET=change-this-to-a-secure-random-string-minimum-32-chars
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

### Step 2: Initialize Database (1 minute)
```bash
npm run seed
```
This creates admin user: `admin@nexuspro.com` / `Admin@123`

### Step 3: Start Development (10 seconds)
```bash
npm run dev
```
Open: http://localhost:3000

**Admin Login**: http://localhost:3000/admin/login

## 🎨 Customization Guide

### Change Your Branding
1. Go to `Admin > Settings`
2. Edit company name, logo, colors
3. Update contact information
4. Modify hero text & images

### Add Your Services
1. `Admin > Services > Add New`
2. Title, description, icon, price
3. Mark as featured to show on homepage

### Create Your Portfolio
1. `Admin > Projects > Add New`
2. Upload images, add description
3. Mark as "Published" & "Featured" as needed
4. Auto-appears on frontend

### Launch Promotions
1. `Admin > Offers > Add New`
2. Set start/end dates
3. Add discount, badge, CTA
4. Enable countdown timer
5. Auto-expires when date passes

### Write Blog Posts
1. `Admin > Blog > Add New`
2. Write article, add category
3. Upload thumbnail image
4. Mark as Published
5. SEO-optimized, auto-appears

### Manage Leads
1. `Admin > Inquiries`
2. See all form submissions
3. Mark as read/unread
4. Track contact information
